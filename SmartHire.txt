SmartHire – AI Based College Placement & Skill Development Platform

Project Specification Document

1. Project Title

SmartHire: Intelligent College Placement and Skill Enhancement Platform

2. Project Overview

SmartHire is a full-stack web application designed to automate and modernize the college placement process.

The platform connects students and companies through an intelligent system that evaluates:

Technical skills

Coding performance

Job requirements

The system is controlled by the college administration to ensure transparency, fairness, and data security.

The platform also includes a coding practice system, smart job matching, real-time communication, and placement management tools.

4. System Users

The system has three types of users.

4.1 Admin (College Authority)

The college administration manages the entire system.

Responsibilities:

Approve student registrations

Approve company registrations

Manage job postings

Monitor coding scores

Schedule interviews

View analytics and reports

4.2 Student

Students use the platform to develop skills and apply for jobs.

Students can:

Register and login

Create and update profile

Add technical skills

Upload resume

Practice coding

Take scored coding tests

View coding scores

Apply for jobs

View interview schedule

Chat with companies

Student Profile Includes

Personal details

Technical skills

Interview status

4.3 Company

Companies use the platform to recruit students.

Companies can:

Register and login

Create company profile

Post job openings

Two modes are available.

Practice Mode

Results are not recorded

Used for skill improvement

Score Mode

Purpose: evaluation.

Features:

Companies can view coding scores

Difficulty Levels

6. Smart Job Matching System

SmartHire automatically identifies eligible candidates.

When a company posts a job, it specifies:

required skills

minimum coding score

interview date

The system checks student profiles.

Eligibility condition:

Student has required skills
AND
Student coding score >= minimum score

If conditions are satisfied:

student is marked eligible

interview date is assigned

notification is sent

7. Job Application System
